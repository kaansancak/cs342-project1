## How to run?

1) Just run 'make' command from terminal

2) For prime `'./prime <N> <M>'`
	* N is number of children and 
	* M is number of processes
	* Max N value: 1 000 000 
	* Max M value: 50

2) For mqprime `'./mqprime <N> <M>'`
	* N is number of children and 
	* M is number of processes
	* Max N value: 1 000 000 
	* Max M value: 5
